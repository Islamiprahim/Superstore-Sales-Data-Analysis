# Superstore Sales Data Analysis & Interactive Dashboard

## Project Overview
This project focuses on transforming raw, disconnected retail transactional data from a Superstore into a refined, dynamic, and actionable visual asset. The main objective is to empower management and business stakeholders to quickly pinpoint high-profit zones, track regional performance, and optimize future sales strategies.

## Technical Features & Solutions Implemented
* **Data Cleaning & ETL:** Handled missing records, eliminated layout discrepancies, and properly formatted metrics using Power Query.
* **Dynamic Visualization:** Engineered interactive Pivot Charts, a tailored dual-axis combo chart for trends, and geographic map analysis to track sales by territory.
* **User-Driven Insights:** Integrated synchronized multi-functional Slicers (Regional and segment filters) for cross-filtering capabilities.
* **KPI Header Cards:** Designed clear, high-level summary metrics showcasing Total Revenue ($2.29M) and Total Profits ($286K) at a glance.

## Business Impact
By replacing cluttered spreadsheets with a cohesive, professional layout, this dashboard eliminates hours of manual data slicing, giving immediate clarity to sales performance and profit margins across different demographics.
